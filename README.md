# Planetary Gear System Design and Assembly

![Planetary Gear Assembly](imeges/plantery-gear)

## Overview
This project presents the **design and mechanical assembly of a planetary gear system**.  
All gears and structural components were designed from scratch and assembled into a complete functional gear mechanism.

The system consists of a sun gear, multiple planet gears, an internal ring gear, and a carrier structure.

## Project Purpose
The objective of this project is to:
- Understand the mechanical structure of planetary gear systems.
- Practice gear design and tooth alignment.
- Assemble multiple rotating components with correct constraints.
- Study motion transmission and torque distribution principles.

This project was completed for learning and mechanical design practice purposes.

## System Components
- Sun Gear (central gear)
- Planet Gears (three equally spaced gears)
- Ring Gear (internal gear)
- Carrier (planet support structure)
- Shafts and mounting elements

## Assembly Scope
- Custom design of all gears and structural parts.
- Full mechanical assembly of the gear system.
- Focus on gear meshing accuracy and rotational alignment.
- No physical manufacturing or load testing performed.

## Mechanical Principle
Planetary gear systems distribute torque across multiple planet gears, allowing:
- Compact design
- High torque transmission
- Smooth rotational motion
